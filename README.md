# Windows Advanced Audit Policy

Набор `.bat` скриптов для настройки расширенного аудита ОС семейства Windows в соответствии с официальными рекомендациями контролирующих органов. Решают проблему с настройкой аудита на локализованных сборках ОС.

## Скрипты

Скрипт

apply_audit_policy_DC.bat -  Контроллеры домена 
apply_audit_policy_Servers.bat - Серверы
apply_audit_policy_workstation.bat - Рабочие станции

## Проблема

Стандартный механизм применения политик аудита через GPO использует файл `audit.csv`. На локализированных версиях Windows этот файл генерируется с лишними запятыми, что приводит к некорректному парсингу и сбоям при применении политик — часть категорий аудита не настраивается.

## Решение

Скрипты обходят нативный GPO-механизм и применяют расширенные политики аудита напрямую через утилиту `auditpol.exe`, не завися от формата `audit.csv`.

## Требования

- Windows 7 / Windows Server 2008 R2 и новее
- Запуск от имени администратора

## Использование

Запустить нужный скрипт в зависимости от роли машины:

```cmd
apply_audit_policy_DC.bat          # Контроллер домена
apply_audit_policy_Servers.bat     # Сервер
apply_audit_policy_workstation.bat # Рабочая станция
```
